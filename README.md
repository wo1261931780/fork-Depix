# fork-Depix

从像素化截图中恢复密码。

## 简介

这是 [Depix](https://github.com/beurtschipper/Depix) 的 Fork 版本。

Depix 是一个用于从像素化截图中恢复密码的工具。它通过匹配像素化字符与已知字符集来还原原始文本。

## 功能

- 从像素化截图中恢复密码
- 支持多种像素化算法
- 命令行界面

## 使用方法

```bash
python depix.py -p /path/to/pixelated-image.png -s images/searchimages.png
```
